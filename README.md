## UniverseSingularity specs / functionality

### Minting Features

- On-chain metadata JSON
- Multi-asset NFT
- Additional supporting assets for NFT
- License files per NFT
- Allow creator to add new assets and contextual secondary assets (and in bulk)
- Allow creator and owner to update default displaying NFT
- Backup assets for IPFS/Arweave redundancy
- Modifiable metadata (if marked, else permanent)
- Allow creator to specify their own external URIs for an NFT asset
- Allow creator to use iFramed animation_url for NFT
- Existing royalty standards with dynamic time-dependent changing values (linear change or specific date hard change)
